# Outbox pattern

## Prerequisites

- [↑ .NET 9](https://dotnet.microsoft.com/en-us/download/dotnet/9.0)
- [↑ Docker Desktop](https://docs.docker.com/desktop/)

## Run

Run API project:

```bash
dotnet run --project src/Outbox.Api
```

Run rocessor project:

```bash
dotnet run --project src/Outbox.Processor
```
